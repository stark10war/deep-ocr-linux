Document Detection

A Tensorflow RCNN model is trained to detect ID cards/documents even in complex and challenging backgrounds.

Document Classification and Adjustments

A YOLO object detection model is trained to detect document specific landmarks to classify documents as PAN, AADHAR, Driving License, Bank Statement etc. Document orientation is also adjusted with Computer Visiontechniques.

Entity Detection with YOLO

Seperate YOLO models are trained for each type of document to detect and classify various entites in the document like Name, ID number, DOB etc. These entites are cropped, prerpocessed and passed to Tessaract OCR engine to get text data.
